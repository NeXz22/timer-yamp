What is mob-programming?

Mob programming is a software development technique in which the entire team works on the same task at the same time, using the same computer and screen. One team member, called the "driver," writes code while the rest of the team, called the "navigators," provide guidance and assistance. The driver and navigators switch roles frequently, with the goal of collective ownership and shared understanding of the code. Mob programming can help teams work more efficiently and effectively by leveraging the collective knowledge and skills of all team members. It can also help improve communication and collaboration within the team.

What does YAMP do?

YAMP tries to migrate the mob-programming technique to a remote work environment. YAMP offers the possibility to track time, set and track goals, and assign roles to participants. By providing a centralized platform for mob-programming, YAMP can help teams practicing mob programming and work more effectively and efficiently. Whether you're just starting out with mob programming or have been using this technique for some time, YAMP can be a valuable resource for your team.
